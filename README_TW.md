硬核實驗室（Gear Core Lab）是一個專業的數碼DIY電腦硬體社區平台，致力於為硬體愛好者提供配置管理、技術交流和科技資訊服務。平台整合了社區論壇、硬體配置工具、新聞資訊等核心功能，為用戶打造一站式的硬體DIY體驗。

### 專案特色
- 🔧 **智能配置系統**：提供硬體相容性檢測和配置推薦
- 💬 **活躍社區**：支持發帖、評論、點讚等社交功能
- 📰 **科技資訊**：及時更新最新硬體動態和技術資訊
- 👥 **用戶系統**：完善的用戶認證、個人中心和關注功能
- 🛡️ **安全保障**：基於Spring Security的權限管理和安全防護

### ⚙️ 配置步驟
1. **數據庫初始化**
   ```sql
   # 创建数据库
   CREATE DATABASE gearcorelab;
   # 导入数据表结构和初始数据
   source src/main/resources/gearcorelab.sql
   ```

2. **配置檔案修改**
   ```properties
   # 修改 application.properties
   spring.datasource.url=jdbc:mysql://localhost:3306/gearcorelab
   spring.datasource.username=your_username
   spring.datasource.password=your_password
   spring.redis.host=your_redis_host
   ```

3. **啟動應用**
   ```bash
   mvn spring-boot:run
   ```

4. **訪問地址**
   - 應用首頁：http://localhost:8080
   - 管理後台：http://localhost:8080/admin
