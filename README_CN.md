### ⚙️ 配置步骤
1. **数据库初始化**
   ```sql
   # 创建数据库
   CREATE DATABASE gearcorelab;
   # 导入数据表结构和初始数据
   source src/main/resources/gearcorelab.sql
   ```

2. **配置文件修改**
   ```properties
   # 修改 application.properties
   spring.datasource.url=jdbc:mysql://localhost:3306/gearcorelab
   spring.datasource.username=your_username
   spring.datasource.password=your_password
   spring.redis.host=your_redis_host
   ```

3. **启动应用**
   ```bash
   mvn spring-boot:run
   ```

4. **访问地址**
   - 应用首页：http://localhost:8080
   - 管理后台：http://localhost:8080/admin
